# 🌸 Iris Flower Classification

The **Iris Flower Classification** project is a Machine Learning model that classifies iris flowers into different species based on their physical characteristics.

The model predicts the species of an iris flower using features such as **sepal length, sepal width, petal length, and petal width**.

This project demonstrates the use of **supervised machine learning classification algorithms**.

# 🧠 How the Model Works

The project follows these steps:

### 1. Data Collection

The **Iris Dataset** contains measurements of iris flowers including:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target variable:

- Species of iris flower

Classes:

- Iris-setosa
- Iris-versicolor
- Iris-virginica

---

### 2. Data Preprocessing

- Load dataset
- Handle missing values (if any)
- Separate features and target
- Train-test split

---
### 3. Model Training

Machine learning algorithms such as:

- Logistic Regression
- Decision Tree
- K-Nearest Neighbors (KNN)

are used to train the classification model.

---

### 4. Model Evaluation

The model performance is evaluated using:

- Accuracy score
- Confusion matrix
- Classification report
